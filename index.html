Getting Started

Before we can start you'll need to make sure you have access to the uwo cascade server. If you dont already have access please contact the ITG helpdesk, enghelp@uwo.ca, located at the Spencer Engineering Building room 1010A.

Having a basic understanding of HTML and HTML editors is preferable but not needed. If you wish to find out more about HTML please visit www.codecademy.com.

Once you've completed these steps, lets head over to cms.uwo.ca.
